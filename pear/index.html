Routing - Simple and lightweight HTTP Request to Action Handler routing for PHP.

A PHP component that provides a mechanism for defining re-usable "action handlers" and for mapping HTTP requests to those handlers. Consists of HTTP request and HTTP response abstractions, the action handler interface, and classes for defining routes.

Install commandpear install bhcarpenter/Routing
